# 🏛️ IRONHAUS: Brutalist Gym Landing Page

A high-impact, cinematic "Scrollytelling" experience built with **React**, **GSAP**, and **Lenis**. This project pushes the boundaries of modern web design with a brutalist, high-contrast aesthetic, scroll-linked parallax typography, and layered reveal mechanics.

## ⚡ Key Features

### 1. 🏗️ Brutalist Poster Architecture
The second page features a "Kinematic Typography" poster. It uses **Horizontal Parallax Scrolling** where massive background watermarks move at different speeds than the central gym interior, creating a 3D illusion of depth.

### 2. 🎞️ Cinematic Scrollytelling
- **Scroll Hijacking & Scrubbing:** Vertical scroll energy is mapped to horizontal movement and animation timelines using **GSAP ScrollTrigger**.
- **Layered Shutter Reveal:** Sections stack and overwrite each other like a physical card deck, creating a high-end "shutter" effect.

### 3. 🌐 Gym Virtual Gallery
A pinned horizontal gallery showcasing remote coaching and live technique masterclasses with high-resolution, brand-aligned imagery ("Curl Corner" and "Red Light Gym").

### 4. 🧭 Precision Navigation
Custom-built navbar with seamless internal scrolling to dedicated sections: **Method**, **Performance**, **Virtual**, and **Training**.

## 🛠️ Tech Stack

- **Core:** React 18, TypeScript, Vite
- **Animation:** GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Smooth Scrolling:** Lenis (Studio Freight)
- **Styling:** Tailwind CSS (Vanilla CSS for custom brutalist components)
- **Aesthetics:** Monumental Typography, Negative Letter Spacing, High-Contrast Palette (Orange, Gold, Charcoal).

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/santhosh2005-santhosh2005/Gym_Landing.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 📐 Design Philosophy
IRONHAUS follows the **Brutalist Design System**:
- **Typography:** geometric, heavy neo-grotesk (Weight 900+).
- **Negative Tracking:** Letters almost overlap for maximum impact.
- **Grids:** Exposed neobrutalist grid systems.
- **Motion:** Intentional, heavy, and physics-based.
